# Thutaytana

Thutaytana is an AI-agent based tool that assists researchers automate the processes of creating posters and presentation slides for the research conferences.

### Some Guidelines
Users are allowed to generate research abstract and poster.

# Steps 
- User input
- Reading research context (image + text)
- Generate research abstract from research context + detailed analysis of images
- Creating a poster based on parsed research context + captions of images
- Give feedback or edit on poster
- Creating presentation slides
